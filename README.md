# MovieCatalog

Website with a movie catalog.

## Technologies

- [Vite](https://vitejs.dev/)
- [React](https://reactjs.org/)
- [React Router Dom](https://reactrouter.com/)

## Installation and setup

In the project directory, you can run:
**npm install**
**npm install react-router-dom**
Run the project:
**npm run dev**
